IDM 241 Alpha

Original interface

Trigger

  • Click on hamburger menu lines, visible on mobile view
  • To close menu, click X icon

Rules

  • Click on hamburger menu lines, visible on mobile view
  • Background color fills page from top to bottom
  • Menu items fade in
  • Hamburger lines change to X symbol
  • When X is clicked, the menu screen slides out from bottom to top

Feedback

  • Background color fills page from top to bottom
  • Menu items fade in
  • Hamburger lines change to X symbol
  • When X is clicked, the menu screen slides out from bottom to top

Loops/modes

  • If the user is on a page listed in the menu items, the text is underlined
