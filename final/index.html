IDM 241 Final Description

Original interface

Triggers

  • Hover on/off menu icon
  • Click menu icon
  • Hover on/off menu items
  • Click menu item
  • Hover on/off "X" button
  • Click "X" button

Rules

  • On clickable item hover, cursor changes to pointer and item scales up in size by 10%
  • Off clickable item hover, cursor-pointer changes to default cursor and item scales down in size by 10%
  • On menu icon and menu item hover, color changes to blue
  • Off menu icon and menu item hover, color changes to black
  • On "X" button hover, color changes to red and icon shakes
  • Off "X" button hover, color changes to black and icon stops shaking
  • On menu icon click, background color fills page from top to bottom
  • On menu item click, a line appears underneath the item from left to right and background color changes. After 1 second delay, new page loads
  • On "X" icon click, menu slides out from bottom to top

Loops/modes

  • Mode: If page is active, the menu item is underlined and the associated background color is visible.
